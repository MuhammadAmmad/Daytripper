Meetup

For Meetup queries, include keywords "meetup events" or "meetup groups" in the query. See these examples.
  • business meetup events
  • technology meetup groups in San Francisco, CA US
  • beer meetup events in Dublin, IE

Filtering

Queries may be filtered by location. If no location is specified then the phone's location is used as default. Please use grammatically correct phrases, e.g. enter "restaurants in New York", not "restaurants New York". For best results, please enter city, state or province (if applicable), and country, e.g. enter "buffet in Las Vegas, NV US", not "buffet in Las Vegas". To specify US states, use 2-letter state codes (a list could be found at this link http://en.wikipedia.org/wiki/ISO_3166-2:US). To specify Canada provinces, use 2-letter province codes (a list could be found at this link http://en.wikipedia.org/wiki/ISO_3166-2:CA). To specify countries, try using 2-letter country code first in the query. If no results are found then try another query using the full country name. A list of country codes could be found at this link http://en.wikipedia.org/wiki/ISO_3166-1#Current_codes.

Similarity Filtering (Seatgeek Only)

A Seatgeek search can find performers similar to a particular performer. For this search a postal code must be used as location filter rather than city, state/province, and country..

Sorting (Yelp Only)

Yelp searches may be sorted by ratings or distance. Please use grammatically correct phrases, e.g. enter "restaurants by distance", not "restaurants distance".
